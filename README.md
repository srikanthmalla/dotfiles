# dotfiles
clone the repo to $HOME

``git clone https://github.com/srikanthmalla/dotfiles ~/``

clone oh-my-zsh

``git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh``

copy zshrc to $HOME

``cp ~/dotfiles/.zshrc ~/.zshrc``

change shell to zsh
``chsh -s /bin/zsh``

copy vimrc to $HOME

``cp ~/dotfiles/.vimrc ~/.vimrc``
